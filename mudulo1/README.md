# 📚 Conteúdo Estudado
## Módulo 1: Fundamentos, Estruturas e POO

- Fundamentos de Python: Tipagem, variáveis, características da linguagem e interação com o usuário.

- Estruturas de Dados: Manipulação de strings, listas, tuplas e dicionários.

- Estruturas Avançadas: Conceitos e aplicações práticas de Pilhas (LIFO) e Filas (FIFO).

- Fluxos de Controle: Estruturas condicionais (if, elif, else) e laços de repetição (for e while).

- Programação Funcional & Tratamento de Erros: Funções puras, paradigmas, map(), filter() e blocos try/except/finally.

- Programação Orientada a Objetos (POO): Classes, objetos, métodos, encapsulamento (atributos privados), herança e polimorfismo.

- Módulos e Pacotes: Organização e reutilização de código.
